# 트레이딩 봇 프로젝트 — Claude Code 컨텍스트

## 나에 대해
- 전업 트레이더 전환을 목표로 하는 코인 선물 트레이더
- 바이비트 선물 실매매 진행 중 (USDT 마진)
- 최종 목표: 퀀트 트레이딩 + 개인 수준의 수학적 트레이딩 모델 구성
- 현재 확률/통계 학습 중, 코딩 초보 → 이해하면서 진행하는 것이 중요

## 거래소
- 바이비트: 선물 (실매매 중) ← 현재 봇 주요 타겟

## 기술 스택
- 알림봇: TypeScript (Node.js)
- 수학/통계 모델: Python (추후)
- 알림: Telegram Bot API
- 환경변수: dotenv
